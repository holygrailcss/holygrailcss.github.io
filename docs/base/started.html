This page is an overview of the Holygrail documentation and related resources.

Holygrail is a CSS and html library for building user interfaces. Learn what Holygrail is all about on our homepage .

Personalizar nuestro Holygrail es muy rápido 





// Importamos fuentes





$font-family-light: 'Fuente-Light-Pro';
$font-family-regular: 'Fuente-Regular-Pro';
$font-family-medium: 'Fuente-Medium-Pro';
$font-family-bold: 'Fuente-Bold-Pro';

$font-family-secondary:'Times new roman';

// Importamos tantas font face como necesitemos 

@font-face {
  font-family: $font-family-bold;
  src: url('../assets/fonts/basis/fuente-Bold-Pro.eot');
  src: url('../assets/fonts/basis/fuente-Bold-Pro.eot?#iefix') format('embedded-opentype'),
  url('../assets/fonts/basis/fuente-Bold-Pro.woff2') format('woff2'),
  url('../assets/fonts/basis/fuente-Bold-Pro.woff') format('woff'),
  url('../assets/fonts/basis/fuente-Bold-Pro.ttf') format('truetype'),
}


//Actualizamos las variables globales

$totalCols :12;

$padding-button: 8px;
